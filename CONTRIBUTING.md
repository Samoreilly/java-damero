# Contributing to Kafka Damero

Thanks for your interest in contributing! This project is still early in development and contributions are welcome.

## How to Contribute

1. Fork the repository
2. Create a branch for your feature or fix
3. Make your changes
4. Test your changes
5. Submit a pull request

## What to Contribute

- Bug fixes
- Documentation improvements
- New features (check PROJECT_IDEAS.txt for ideas)
- Test coverage improvements
- Performance optimizations
- Examples and use cases

## Getting Started

1. Clone the repository
2. Make sure you have Java 21+ and Maven installed
3. Run `mvn clean install` to build the project
4. Check out the example-app for usage examples

## Questions?

Feel free to open an issue if you have questions or want to discuss a feature before contributing.
